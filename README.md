# Event Hub React Project

A React-based event management platform.

## Prerequisites

- Node.js (v14 or higher)
- npm or yarn

## Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd event-hub-react-project
```

2. Install dependencies:
```bash
npm install
# or
yarn install
```

## Development

Run the development server:
```bash
npm start
# or
yarn start
```

The application will open at [http://localhost:3000](http://localhost:3000)

## Build

Create a production build:
```bash
npm run build
# or
yarn build
```
